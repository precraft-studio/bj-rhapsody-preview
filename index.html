App 描述 · Description

你真的了解 21 點嗎？ 每個玩家心裡都有些疑問： 「這把該補牌還是停牌？」 「連贏五把該加注嗎？」 「算牌到底有沒有用？」 BLACKJACK Rhapsody 是一套為認真玩家設計的 21 點策略研究工具，讓你不靠運氣、用數據找到答案。 ── 7 種模式，全方位訓練 ──【正常模式】 標準 21 點遊玩。每手即時顯示 Running Count 與最佳策略建議，邊玩邊學，快速建立直覺。 【闖關模式】 Parlay 連勝策略實戰。設定目標關數，視覺化追蹤每一關的進度與損益，感受連勝累積的張力。 【馬丁格爾模式】 馬丁格爾倍投策略追蹤。自動計算每次下注金額，記錄連敗極值與歷史高點，清楚看見策略的風險邊界。 【策略設置】 完整編輯你的 Hard / Soft / Pairs / Insurance 策略矩陣。設定 12 個 RC 區間的自訂投注金額，打造屬於你的算牌投注系統。 【模擬分析】 一鍵執行最高 200 萬手模擬。驗證長期勝率、EV%（期望值）、損益曲線，以及各 RC 區間的表現差異。4 種投注模式均可模擬：固定下注 / 自訂 RC 投注 / 闖關 / 馬丁格爾。 【場次模擬】 以「到場次數」為單位的資金曲線模擬。觀察多次賭場到場的損益分布、破產風險與最大回撤，評估長期策略的穩定性。 【算牌練習】 Hi-Lo 系統訓練，LV1 至 LV5 難度遞增： • LV1：閃卡單張計數 • LV2：玩家單手計數 • LV3：玩家＋莊家（全明牌） • LV4：玩家＋莊家累計 RC 追蹤 • LV5：多玩家＋莊家累計 RC 追蹤 ── 完整賭場環境設定 ──12 項賭場規則完整可調： • 牌組數（1 / 2 / 4 / 6 / 8 副） • 洗牌滲透率 • 莊家行為（S17 / H17） • BJ 賠率（3:2 / 6:5 / 1:1） • 投降規則（Late Surrender / 不允許） • 保險規則（含 Even Money） • 加倍限制（任意 / 9-11 / 10-11） • 分牌後加倍（DAS / NDAS） • 分牌上限（2 / 3 / 4 手） • 分 AA 後補牌規則 • 分 AA 後可再分 • 分牌後 21 點賠率（1:1 / 依 BJ 賠率） ── 其他特色 ──• EV% 熱圖矩陣：視覺化呈現各手牌組合的期望值差異 • RC 分布表：各計數區間的勝率、平均下注、損益一覽 • 30 種語言支援 • 14 套色彩主題（深色 8 套、淺色 6 套） • 數據持久化：關閉後重開，所有設定與統計完整保留 ── 適合誰使用？──✓ 想了解 21 點基本策略的初學者 ✓ 正在練習 Hi-Lo 算牌的進階玩家 ✓ 想驗證長期投注策略效果的研究型玩家 ✓ 出發賭場前想熟悉規則的旅遊玩家 本 App 為策略研究與練習工具，不涉及真實金錢交易。 BLACKJACK Rhapsody — 用數據，讀懂 21 點。
